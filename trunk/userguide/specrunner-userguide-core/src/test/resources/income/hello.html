Using custom with properties file

Using class with property file
Variable global previously defined is available. (${name})

You can use class with a name of a class mapped in plugin_css.properties . Other class values can be used in the same attribute. . 

                    hello=example.core.PluginHello

Check if message is 'Hello ${name}'.

Using defined tag with property file
You can use a specific tag <hello> mapped in plugin_element.properties . 
                    hello=example.core.PluginHello
