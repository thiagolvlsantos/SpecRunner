Exemplo de drag and drop com JQuery

Para o teste é necessário abrir um navegador (tamanho de 600x400 para facilitar visualização). Considerando também que o diretório de testes é o 'src/test/resources/income'.

Abra o arquivo example.html (baseado no exemplo em JQuery). Agora realize o movimento de Drag&Drop sugerido, verificando que o texto apresentado no alerta é 'dropped', confirme o alerta.

Por fim finalize o navegador.
