Import escondido

Ao importar A e B a lista importada é [example.imports.a,example.imports.b]

Import visível

Importando diretamente o pacote example.imports.c a lista importada passa a ser [example.imports.a,example.imports.b,example.imports.c]

Import em colunas

Importando colunas
example.imports.d example.imports.e
a lista importada passa a ser [example.imports.a,example.imports.b,example.imports.c,example.imports.d,example.imports.e]

Import em linhas

Finalmente importando linhas
example.imports.f
example.imports.g
a lista importada passa a ser [example.imports.a,example.imports.b,example.imports.c,example.imports.d,example.imports.e,example.imports.f,example.imports.g]
