Partial Matches

Username searches return partial matches, i.e. all usernames containing the search string are returned.

Example

Given these users:

FirstLast
johnlennon
ringostarr
georgeharrison
paulmccartney

Searching for "arr" will return:

Matching Usernames
george.harrison
ringo.starr
Matching FirstMatching last
georgeharrison
ringostarr
